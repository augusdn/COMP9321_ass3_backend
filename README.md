### How do I get set up? ###

* Create virtual env: 
`python3 -m venv ass3_demo`
* Activate virtual env: 
`source ass3_demo/bin/activate`
* Install required libraries: 
`pip install -r requirements.txt`
* Run API app: 
`python3 ass3-API.py`
